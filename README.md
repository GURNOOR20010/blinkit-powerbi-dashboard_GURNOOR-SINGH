# 🛒 Blinkit Sales Dashboard

A Power BI dashboard project that analyzes Blinkit's sales performance, customer preferences, outlet distribution, and product categories. This dashboard provides actionable insights through interactive visualizations and KPIs.

## 🚀 Project Overview

The Blinkit Sales Dashboard is designed to help businesses understand:

- Total Sales Performance
- Average Sales per Item
- Product Ratings
- Outlet Distribution
- Fat Content Analysis
- Item Type Performance
- Outlet Size & Location Analysis

The dashboard enables data-driven decision-making through clear and interactive visualizations.

## 📌 Insights Generated

### Sales by Outlet Establishment Year
- Highest sales recorded in 2018.
- Consistent growth observed across most years.

### Outlet Size Analysis
- Medium-sized outlets contribute the highest sales.
- Small and High-sized outlets contribute comparatively less.

### Outlet Location Analysis
- Tier 3 locations generate the highest revenue.
- Tier 1 locations show the lowest sales among all tiers.

### Item Type Performance
Top-performing categories include:
- Fruits & Vegetables
- Snack Foods
- Household Items
- Frozen Foods
